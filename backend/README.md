# Back-End README.md

To run the Back-End server successfully:

- Make sure that you are in the backend folder -> open terminal -> cd backend
- Message [me](https://wa.me/201009977022?text=Hi%20Abdullah%2C%20please%20send%20me%20the%20.env%20file) to get the `.env` file, then put it in the backend folder (make sure that it is named ".env")
- Install Dependencies -> `npm i`
- Run the Back-End server -> `npm run server`
